CLASSES
MyApp: Constructs and makes the JFrame visible. 
GameCanvas: Carries out computation and draws the game.
HighScore: Defines methods to store highscores in a file.

LOGIC
The ball follows projectile motion using the projectile motion formulas.

INSTRUCTIONS 
Try to keep the ball from touching the bottom. To start the game, press the SPACE bar. Use the LEFT and the RIGHT direction button to move the pad. After every level, the size of the bar decreases and the time for the level increases. The current score, level, highscore, number of lives, and the timer are displayed on the top of the game.

RUN
To run the code, run the MyApp file using an IDE or from the terminal by running "javac MyApp.java" followed by "java MyApp".
